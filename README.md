# Sistema de Gestión – Laravel 12 + React

Proyecto desarrollado como prueba técnica para la posición de **Desarrollador Laravel**.  
El sistema implementa la gestión completa de productos, clientes, inventario y pedidos, utilizando **Laravel 12** como backend y **React + Vite** como frontend.

---

## Descripción General

La aplicación permite registrar y administrar productos, clientes y pedidos de manera eficiente, manteniendo actualizadas las existencias en el inventario.  
Incluye validaciones de datos, control de stock y una interfaz moderna construida con React y Tailwind CSS.

---

## Características Principales

- CRUD completo de productos y clientes  
- Gestión de inventario con ajuste manual de stock  
- Registro de pedidos con múltiples productos  
- Descuento automático de inventario al crear un pedido  
- Validación de stock insuficiente antes de registrar pedidos  
- API REST funcional con endpoints estructurados  
- Validaciones en formularios con mensajes descriptivos  
- Interfaz responsiva y fácil de usar  

---

## Requisitos del Sistema

- PHP >= 8.2  
- Composer  
- Node.js >= 18.x  
- npm o yarn  
- MySQL como base de datos
